
Projeto de Análise de Dados com foco em Segurança de Rede
Recentemente desenvolvi um projeto prático utilizando Power BI e Python para simular e analisar eventos de rede.

O objetivo foi entender como dados podem ser utilizados para identificar padrões de tráfego e possíveis comportamentos suspeitos.

📊 No dashboard, analisei:
✔ Total de eventos de rede
✔ Quantidade de eventos bloqueados
✔ Percentual de bloqueios (indicador de risco)
✔ Uso de portas na rede
✔ Distribuição de protocolos
✔ Interpretação dos dados para tomada de decisão

📌 Um dos principais pontos da análise foi observar que aproximadamente 16% dos eventos foram bloqueados, indicando a presença de atividade suspeita, embora ainda dentro de um nível considerado moderado.

🧠 Esse projeto me ajudou a evoluir na prática em:
▪ Análise de dados
▪ Power BI (visualização)
▪ Python (simulação de dados)
▪ Interpretação de informações voltadas à segurança
